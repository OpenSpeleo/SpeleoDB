Tool: Mnemo DMP Doctor

This tool helps you correct an Mnemo Survey file after a dive if you mis-calibrated or mis-managed your Mnemo device. Instead of re-diving, you can apply corrections to fix common survey errors.

Note: This tool does not store any of the data sent. Please back up your data and result.

Fix Corrupted DMP Files

This feature automatically repairs corrupted or malformed Mnemo DMP files that fail to import properly into Ariane survey software.

Enable this option if your DMP file won't import into Ariane or shows errors during the import process.

No Yes

About Date Correction

Change the survey date if it was incorrectly set on your Mnemo device.

This is useful when you forgot to update the date on your device or if the date was lost due to battery issues.

About Shot Length Correction

Correct systematic measurement errors in shot lengths due to calibration issues.

Example: If you measured a known 100ft distance and the Mnemo recorded 104ft, calculate the scaling factor:

(100ft ÷ 104ft) × 100 = 96.15%

This will proportionally scale all shot lengths in your survey.

%

About Compass Correction

Adjust for compass calibration errors or magnetic interference.

Offset Example: If a passage should be 190° but was measured as 205°, apply an offset: 190° - 205° = -15°

This offset will be applied to all compass readings. Positive values rotate right, negative values rotate left.

Reverse Direction: Use this to fix surveys done in the wrong direction (In vs Out). This applies a 180° compass offset, essentially flipping the direction of the entire survey.

°

About Depth Correction

Compensate for depth calibration errors in your Mnemo.

Example: If at 100ft actual depth the Mnemo measured 104ft, calculate the offset: 100ft - 104ft = -4ft

Positive offsets make all depths deeper, negative offsets make them shallower.

ft
