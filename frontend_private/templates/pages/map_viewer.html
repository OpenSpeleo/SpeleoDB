Map Interaction Guide

🖱️ Right-Click Menu Options

On Empty Map Area:
  • • Create Station - Add a new station at the clicked location
On Existing Station:
  • • View Station Details - Open the station management panel
  • • Edit Station - Modify station information (with write access)
  • • Delete Station - Remove the station and all its resources (with write access)

🎯 Mouse Actions

Left Click: Select and view station details
Right Click: Open context menu for actions
Drag & Drop: Move stations to new locations (with write access)

⌨️ Keyboard Shortcuts

ESC Close modals and cancel operations
Enter Confirm actions in modals

💡 Pro Tips

  • • Stations appear when zoomed in (zoom level 14+)
  • • Use the project panel to show/hide different survey projects
  • • Station names must be unique within each project
  • • All actions are saved automatically
